🛡️ Problemas de Conexão? (Ajuda)

Se o celular não conseguir carregar as páginas, o Windows pode estar bloqueando a rede. Execute o comando abaixo no PowerShell como Administrador para liberar o sistema:

New-NetFirewallRule -DisplayName "TrayoFREE" -Direction Inbound -LocalPort 3000 -Protocol TCP -Action Allow

Atenção: Certifique-se de que o computador e os celulares estão conectados na mesma rede Wi-Fi.
